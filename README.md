# EZ
## Swimming Pool Party Website with Inbuilt Booking System
This is a website for booking swimming pool parties. It features an inbuilt booking system that allows users to select the date for their party, as well as the number of guests and any additional services they require.

### Features
* Booking system: Users can select the date and time for their party, as well as the number of guests and any additional services they require.
* Email notifications: Users receive email notifications confirming their booking and payment.
* Contact form: Users can submit a contact form to get in touch with the website owner for any questions or concerns.
* Responsive design: The website is designed to be responsive and work well on desktop and mobile devices.
